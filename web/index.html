1. How does Gishath Fetch work?
Gishath Fetch searches the different local game stores' (LGS) website for the card name entered, performs filtering of result for higher accuracy and returns the compiled result sorted by price.
2. Why is Gishath Fetch slow?

Different LGSs' websites have varying response times. The overall search time will be held back by the longest/slowest one.
