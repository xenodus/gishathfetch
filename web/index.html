FAQ / Help
  1. How does Gishath Fetch work?
  2. Can I search for specific quality (e.g. NM/LP) of card?
  3. Why is Gishath Fetch so slow?
  4. Is Gishath Fetch free to use?
1. How does Gishath Fetch work?
Gishath Fetch searches the different local game stores' (LGS) website for the card name entered and returns the compiled result sorted by price.
2. Can I search for a specific quality (e.g. NM/LP) or version (e.g. extended art) of card?
No (for now).
3. Why is Gishath Fetch so slow?

Gishath Fetch searches the selected LGSs concurrently. However, different LGSs' websites have varying response times. The overall search time will hence be held back by the longest one.
